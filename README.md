# This is a decentralized application for securely storing and managing
# criminal records using blockchain technology.
# The system ensures data integrity, transparency, and tamper resistance.

# =========================================================
# FEATURES
# =========================================================
# - Decentralized blockchain storage
# - Proof of Authority (PoA) consensus
# - SHA-256 and RSA based security
# - Flask based web application
# - Multiple authority nodes
# - Immutable criminal records

# =========================================================
# PROJECT OVERVIEW
# =========================================================
# Traditional criminal record systems are centralized and vulnerable.
# This project uses a private blockchain where only authorized
# authorities can validate and store records securely.

# =========================================================
# TECHNOLOGIES USED
# =========================================================
# - Python
# - Flask
# - Private Blockchain
# - Proof of Authority (PoA)
# - SHA-256
# - RSA Cryptography
# - JSON

# =========================================================
# INSTALLATION & SETUP
# =========================================================

# Step 1: Clone the repository
git clone https://github.com/harshitak4/CriminalRecordManagementSystem.git

# Step 2: Go into the project folder
cd CriminalRecordManagementSystem

# Step 3 (Optional): Create a virtual environment
python -m venv venv

# Step 4: Activate virtual environment
# Linux / Mac:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# Step 5: Install dependencies
pip install -r requirements.txt

# Step 6: Run the application
python main.py

# =========================================================
# USAGE
# =========================================================
# - Run main.py
# - Open browser at http://localhost:5000
# - Add and verify criminal records
# - Authority nodes validate records using PoA
